OBS = Pesquise todas as perguntas na Internet.

- O que é API? 
-- Aplication Programming Interface (Interface de Programação de Aplicações)

- Webservices = Serviços da Web

- O que é REST? 
-- Representational State Transfer (Transferência de Estado Representational)

- O que é RESTful?
-- Sistema que está usando REST

________________________________________________________________________________________

Padrão REST:

GET https://meusite.com.br/api/users                                            //pegar informação de todos os usuarios
GET https://meusite.com.br/api/user/10                                          //pegar informação de um usuario especifico 
PUT https://meusite.com.br/api/user/10 (name: 'novoNome', age: 115)             //alterar informação
POST https://meusite.com.br/api/users (name: 'novoNome', age: 115)              //adicionar um usuário novo
DELETE https://meusite.com.br/api/user/10                                       //deletar um usuario





CORS:
Cross-origin Resource Sharing = Troca de recurso de origem diferente
